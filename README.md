FUTURE_PE_03 – AI SEO Blog & Content Cluster System for 369 Fitness Gym (Chennai)
Task Description

This project was created as part of the Future Interns Prompt Engineering Internship – Task 3.

The objective was to design a structured AI prompt system that generates:

SEO-optimized blog posts

Keyword intent mapping

Content cluster strategy

Internal linking architecture

Local SEO-focused content

The system is built for:

Business: 369 Fitness Gym
Location: Chennai
Primary Goals: Transformation & General Fitness
Target Audience: College students, working professionals, general fitness seekers

This project does not simply generate a blog post.
